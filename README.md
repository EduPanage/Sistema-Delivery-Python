# Sistema Delivery Python

Sistema backend simples em **Python** que simula o funcionamento de um aplicativo de delivery.  
Focado em praticar lógica de programação, manipulação de listas/dicionários e estrutura de código modular.

Ideal para estudos de Python básico/intermediário, com operações como cadastro, listagem e remoção de restaurantes, pedidos e itens de menu.

## Funcionalidades Principais

- Cadastro e gerenciamento de restaurantes  
- Listagem de opções disponíveis  
- Remoção de restaurantes (com verificação de existência)  
- Simulação de pedidos e processamento básico  
- Menu interativo no console (opções numeradas)

## Tecnologias Utilizadas

- **Python 3** (padrão)  
- Bibliotecas nativas: `os` (limpeza de tela), manipulação de strings e listas  
- Estrutura modular:  
  - `app.py` → Ponto de entrada e menu principal  
  - `function.py` → Funções reutilizáveis (cadastro, remoção, listagem, etc.)
